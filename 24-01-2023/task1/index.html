Task1 JavaScript Error Handling

Errors are mainly three type - Compilation Error , Runtime Error, Logcal Error

Compilation Error : When compiler try to compile the program and got any syntax error or undefined keywords.

Runtime Error : A runtime error occurs when a program is syntactically correct but contains an issue that is only detected during program execution.

Logical Error : When program compiled and executed but not giving expected result.

  1. Try Catch block:: The try statement allows you to define a block of code to be tested for errors while it is being executed. The catch statement allows you to define a block of code to be executed, if an error occurs in the try block.
  2. Try Catch block and finally:: The finally statement lets you execute code, after try and catch, regardless of the result.
  3. throw Error:: The throw statement allows you to create a custom error. Technically you can throw an exception (throw an error). The exception can be a JavaScript String, a Number, a Boolean or an Object
